About This Chatbot

A premium local-first AI chat workstation built with Flask.

What It Is

ALDEA is designed to feel like a modern operations console: calm, focused, and highly configurable. It supports OpenAI-compatible endpoints and Ollama native chat, so you can point it at local or remote models.

Voice Pipeline

Optional integrations let you add speech both ways: Whisper.cpp for transcription and Coqui-TTS for synthesized output. Keep everything local when privacy and control matter.

Persistence

Conversations can be saved and restored from a chat archive. Settings are stored separately, allowing rapid iteration on prompts, models, and server endpoints.

Design Language

The interface uses a green-black-silver metallic style with adaptive light and dark modes, responsive layouts, and intentionally sharp visual hierarchy for desktop and mobile.
